AutoStream AI Agent

Project Overview

This project is an AI Agent system built using Python and LangChain.
It uses RAG (Retrieval-Augmented Generation) to answer user queries based on stored knowledge.

Features

* AI Agent workflow
* RAG-based response system
* Lead capture functionality
* Modular code structure

Tech Stack
* Python
* LangChain
* JSON (for knowledge base)


Project Structure

* `agent/` → agent logic
* `rag/` → retrieval and QA system
* `tools/` → helper tools
* `data/` → knowledge base
* `app.py` → main entry point


How to Run
```bash
pip install -r requirements.txt
python app.py
```
